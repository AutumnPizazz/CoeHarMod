## 术语 / 简称

有时玩家们为了追求方便，会简化一些称呼，如将巴比伦弓手简称为巴弓、将大工程师简称为大工，这里不多作赘述，仅介绍一些特殊的简称 / 术语。

### 英文简称

| 简称 | 英文全称 | 含义 |
| :-: | :-: | :-: |
| T | Turn | 回合 |
| UU | Unique Unit | 特有单位 |
| UA | Unique Ability | 特有能力 |
| UB | Unique Building | 特有建筑 |
| UI | Unique Improvement | 特有设施 |
| MOD | Modification | 游戏扩展程序，即模组 |
| TJ | 空 | 太监，即还没打完就不玩了 |
| G | Gold | 金，即金钱、现金 |
| AI | artificial intelligence | 人工智能，即电脑玩家 |
| RA | Research Agreement | 科研协议 |
| GPT | Gold Per Turn | 回合金，即每回合获得的金币 |
| SL | Save Load (Ciallo~) | 即保存，读档 |

### 常用术语

| 术语 | 含义 |
| :-: | :-: |
| 爆铺 | 多铺城，靠大量城市发展国力的玩法 |
| 精铺 | 少铺城，专注发展少量优质城市的玩法 |
| 魔王流 | 顶着大量红脸继续推进 |
| 蘑菇 | 指不铺城，光发展的 AI 或玩家，就好像蘑菇一样等人采 |
| 种田 | 专注发展自身国力 |
| 骗宣 | 将手中按回合交付的资源等换 AI 手中的现金后宣战，这样你拿到了现金而 AI 却什么都得不到 |
| 买宣 | 用金钱等交易让 AI 宣战别人 |
| 背刺 | 长期友好或中立后突然敌对或宣战 |
| 开门 | 启用一个政策树，有对应效果 |
| 关门 | 点亮完一个政策树所有的政策，有对应效果 |
| 刮刮乐 | 通过工人不断修复地块设施进行反复劫掠 |
| 勾引大法 | 主动用工人吸引敌军进入某地块后围殴 |
| 自主左二、传统右二等称呼 | 对 xx 政策树左边 / 中间 / 右边第 x 个政策的简称 |
| 锤子 | 产能 |
| 瓶子 | 科研点数 |
| 鸽子 | 信仰点数 |
| 坐城 | 建立城市 |
