## 📝 模组简介

CoeHarMod 大幅扩展了原版 Unciv 的游戏内容，包括：

- 新增多个文明（商朝、周朝、秦朝、明朝、元朝等中国朝代，以及亚特兰蒂斯等）
- 重做总督系统
- 调整万神殿、政策、奇观数值
- 重做宜居度（快乐）系统
- 以及大量平衡性调整...
